# M12: Tracker Integration

**Status:** in flight. Target version: v1.15.0 "Tracker Integration".

Adds an opt-in tracker mode for teams using Linear or Jira as their canonical AC backend. Tracker mode holds only ACs + ticket metadata; everything else (overview, NFRs, technical-spec, testing-spec, plan, traceability, archive) stays in local `specs/`. Bidirectional AC sync via STE-17 with per-AC B-style conflict prompts. Adapters are declarative markdown + TypeScript/Bun helper binaries. `mode: none` (default) preserves pre-M12 behavior byte-for-byte per Pattern 9.

**FRs covered:** STE-8..39 (11 FRs).

**Dependencies:** M11 (latest shipped); M12 builds on the reshaped `/spec-write` per-section wording and the one-at-a-time discipline from v1.14.0.
