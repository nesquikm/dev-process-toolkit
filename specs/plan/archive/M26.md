## Why this milestone exists

The plugin has zero external installs. Every "transition window," "lenient-on-migrated-plans," "pre-M12 backward-compat," and "Linear V1 SSE retirement" surface exists to protect a user base that does not exist. Each item in this milestone removes shipping surface that:

- has no live consumer (e.g., `strip_ulid` migration tool already ran on the maintainer's archive),
- has no live use-case (e.g., 2026-05-11 SSE shutdown — the V1 endpoint is already replaced everywhere),
- frames live content as transitional (e.g., "lenient on round 1 / strict on round 2" gate-check branches that nobody triggers),
- documents migration paths nobody will run (e.g., v1 → v2 layout, none ↔ tracker rollback prompts).

Cumulative reduction estimate: ~600 LOC code + ~300 lines doc prose deleted; ~400 lines doc prose rewritten from transition-state to final-state framing.
