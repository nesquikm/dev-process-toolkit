# STE-369: gate-check plan_ship_coherence probe + /spec-archive --parked {#STE-369}

## Requirement

Archived-but-unshipped milestones must fail `/gate-check` loudly and deterministically, and deliberately parked milestones must stay visible ŌĆö never silent. This is the beacon half of M99: STE-368 makes "shipped" a file property (`shipped_in` frontmatter); this FR makes its *absence* on an archived plan an ERROR, with `ship_state: parked` as the explicit, NOTES-rendered opt-out for milestones intentionally batched on sibling branches (the M92/M97 pattern). Without the probe, forgetting the ceremony is free; with it, the debt surfaces on every gate run.

## Acceptance Criteria

- AC-STE-369.1: **Stamp coherence.** New ERROR-severity gate-check probe `plan_ship_coherence`: for every `specs/plan/archive/M<N>.md` carrying `shipped_in: v<X.Y.Z>`, a `## [X.Y.Z]` heading must exist in `CHANGELOG.md`; a missing heading or malformed stamp value ŌćÆ GATE FAILED (corrupt stamp) with the NFR-10 canonical message naming the plan, the stamp value, and the remedy. {#AC-STE-369.1}
- AC-STE-369.2: **Unshipped debt fails the gate.** An archived plan with neither `shipped_in` nor `ship_state: parked` ŌćÆ GATE FAILED with a typed remedy naming the post-merge ceremony recipe: `/spec-archive M<N>` (only when FRs are still active) ŌåÆ `/ship-milestone M<N>` ŌåÆ `/pr`. {#AC-STE-369.2}
- AC-STE-369.3: **Parked is visible, never silent.** `ship_state: parked` ŌćÆ the probe passes and emits a GATE PASSED WITH NOTES row enumerating every parked milestone. Scope guard: the probe reads `specs/plan/archive/` only ŌĆö live plans under `specs/plan/` are exempt by construction. {#AC-STE-369.3}
- AC-STE-369.4: **`--parked` flag.** `/spec-archive M<N> --parked` writes `ship_state: parked` into the plan's frontmatter during the archival flip, rendered inside the existing mandatory diff-approval gate (no new prompt). Exit hints: default runs end `Archived. Next: /ship-milestone M<N>`; parked runs end `Archived (parked). Unpark by shipping: /ship-milestone M<N>`. {#AC-STE-369.4}
- AC-STE-369.5: **Probe row + tests.** Probe row added to `skills/gate-check/SKILL.md`; `tests/gate-check-plan-ship-coherence.test.ts` covers the positive case plus one negative per violation shape (unstamped-unparked ŌćÆ FAILED, stamp-without-CHANGELOG-header ŌćÆ FAILED, parked ŌćÆ NOTES row); a meta-test pins the `--parked` prose and both exit-hint variants in `skills/spec-archive/SKILL.md`. {#AC-STE-369.5}

## Technical Design

- **Probe** `adapters/_shared/src/plan_ship_coherence.ts`: iterate `specs/plan/archive/*.md`, parse frontmatter, apply the three-way decision matrix (stampedŌåÆcross-check CHANGELOG heading; parkedŌåÆcollect for NOTES; neitherŌåÆFAILED). Pure file reads ŌĆö no git, no network, no LLM judgment; deterministic by construction (survives squash merges, works in consumer projects).
- **Severity model:** violations are ERROR (GATE FAILED); parked milestones render as a single GATE PASSED WITH NOTES row (`parked milestones: M92, M97`) so the opt-out never disappears from view ŌĆö the counter-pressure against parked becoming a reflexive get-to-green stamp.
- **`/spec-archive` changes:** `--parked` parsed alongside existing flags (`--plan-only`); the flip writes `ship_state: parked` next to `status: archived` / `archived_at`; both exit-hint variants are exact-format strings (meta-test-pinned).
- **Remedy text** is a single canonical string shared with `docs/ship-milestone-reference.md`'s recipe section (STE-370) so the two never drift.

## Notes

- **Origin.** /brainstorm 2026-07-06. The state this probe detects is probe-invisible today by construction ŌĆö every existing version signal is bumped only by `/ship-milestone`, so they stale together and #9b passes.
- **Main risk (accepted at design time):** `ship_state: parked` eroding into a reflexive get-to-green stamp; the NOTES-row visibility is the designed counter-pressure. Revisit if parked rows accumulate.
- **Sibling FRs.** Depends on STE-368 (stamp semantics + same-commit backfill). STE-370 surfaces the same debt at `/pr` and bare-`/ship-milestone` time.
