## Requirement

**Description:** The plugin runs skills from its installed marketplace location — users never edit skill files. Remove the manual setup documentation and convert `<!-- ADAPT -->` markers in skill/agent files from "edit this file" comments into runtime LLM instructions.

## Acceptance Criteria

- AC-15.1: `gate-check/SKILL.md` does not contain `<!-- ADAPT: If copying this skill manually`
- AC-15.2: `tdd/SKILL.md` does not contain `<!-- ADAPT: If copying this skill manually`
- AC-15.3: No file in `skills/` or `agents/` contains the literal string `<!-- ADAPT`
- AC-15.4: `gate-check/SKILL.md` domain-specific criteria guidance (previously ADAPT marker at line 47) is rephrased as a plain instruction referencing the project's CLAUDE.md
- AC-15.5: `implement/SKILL.md` spec breakout threshold (previously ADAPT marker at line 115) is rephrased as a plain instruction to check CLAUDE.md for custom threshold, defaulting to 3
- AC-15.6: `implement/SKILL.md` domain-specific checks (previously ADAPT marker at line 170) is rephrased as a plain instruction with inline examples
- AC-15.7: `visual-check/SKILL.md` dev server URL (previously ADAPT marker at line 44) is rephrased as a plain instruction to detect URL from CLAUDE.md or project config
- AC-15.8: `agents/code-reviewer.md` project patterns (previously ADAPT marker at line 13) is rephrased as a plain instruction referencing CLAUDE.md
- AC-15.9: `README.md` does not contain a "## Manual Setup" section
- AC-15.10: `CLAUDE.md` does not contain a "### Manual setup" section
- AC-15.11: `docs/adaptation-guide.md` intro paragraph no longer mentions "manual setup" — reframed as reference for what `/setup` does and how to customize after setup
- AC-15.12: `docs/adaptation-guide.md` Step 1 and Step 3 no longer instruct users to edit skill files directly — reframed as "configure via CLAUDE.md"
- AC-15.13: Template files (`templates/**`) still contain their `<!-- ADAPT -->` markers (unchanged)
- AC-15.14: `README.md` still links to `adaptation-guide.md` but with updated description (no longer "manual setup")
